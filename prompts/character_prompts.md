# Character Prompts

This file documents the character prompts used for the static character generation experiments.

Each prompt was tested across the static workflow configurations listed in the thesis. Each prompt was repeated with five random seeds.

## Shared negative prompt

Negative prompt:
(worst quality, low quality:1), blurry, grainy, deformed, extra limbs, extra fingers, long arms, robotic, photorealistic, shiny skin, 3D render, glowing, gradients, complex background, shadows, text, watermark, signature

## Character prompt 1 - Girl

Positive prompt:
young girl with long blonde hair, wearing a pink t-shirt, tan pants, and white sneakers, standing with legs apart, arms down, facing slightly right, 2D cartoon style, simplistic cartoon style, bold outlines, flat colors, small proportions, dungeon crawler sprite style
