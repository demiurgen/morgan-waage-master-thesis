# Animation Prompts

This file documents the animation prompts used for the animated asset generation experiments.

Each prompt was tested across the animation workflow configurations listed in the thesis. Each prompt was repeated with five random seeds.

All animation outputs were generated at 512x512 pixels with 36 frames at 30 frames per second.
