# Reference Inputs

This file documents the reference inputs used in the workflows.

The reference images and reference videos are not included in this repository because they contain copyrighted material from the Unity reference pack or derived experiment material.

## Static asset workflows

Reference inputs were used in workflows that included ControlNet and IP-Adapter.

### ControlNet inputs

ControlNet reference images were used to guide structure, pose, or general layout.

For character generation, the reference input defined the side-view idle pose.

For background generation, the reference input defined the horizon placement and broad silhouette composition.

For object generation, the reference input defined the object structure and general shape.

### IP-Adapter inputs

IP-Adapter reference images were used to guide visual style.

The reference images were selected from the same Unity reference pack used as the visual style standard in the thesis.

## Animation workflows

Reference inputs were used in image-to-video and video-to-video workflows.

Image-to-video workflows used a still character image as the visual reference.

Video-to-video workflows used a reference motion video to guide the walk cycle movement.
